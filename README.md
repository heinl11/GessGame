# GessGame

A simulated game of Gess, a chess/go variant, coded in Python.  Gess is a two player game and rules can be found [here:](https://www.chessvariants.com/crossover.dir/gess.html)
One player will play as <strong>w</strong> and the opposing player with will play as <strong>b</strong>.  Once a player loses a ring of their own pieces, the game is over.
